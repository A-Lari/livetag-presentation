Introduction

La solution All-in-QR “LiveTag” est un projet de solution tout-en-un facilitant l’organisation lors d’évènements live type salons, conférences, hackathons

Introduction

L’élément central de la solution est la génération d’un QRCode tout en un pour les parties prenantes d’un évènement : participants, conférenciers, exposants

Ce QRCode individuel permettra

  • De contrôler les entrées/sorties individuelles
  • De contrôler l’accès aux activités, conférences, repas, etc...

Besoin / Cahier des charges

  • Une application web permettant la gestion des évènements et la génération de QRCode
  • Une application mobile « compagnon » permettra aux membres du staff d’effectuer les différentes vérifications de contrôle et d’accès

MVP de l'application web

  • Connexion/Inscription
  • Gestion des évènements par utilisateur connecté
  • Gestion des activités par évènement
  • Gestion des rôles par évènement
  • Gestion des participants par évènement

MVP+1 de l'application web

  • [MVP+1] Pour chaque rôle de l’évènement, il doit être possible de générer un lien emmenant vers une page d’inscription
  • Possibilité de s'inscrire à des activités hors roles sur la page d'inscription généré par le lien

MVP+2 de l'application web

  • [MVP + 2] Le paiement pourra être effectué à l’inscription avec Stripe.

MVP de l'application mobile

  • Saisir et modifier le code évènement (généré par l'app web)
  • Scanner un QRCode et de vérifier si le participant est bien inscrit à l’évènement
  • Scanner un QRCode et de vérifier si le participant est bien inscrit à une activité (inclus dans le rôle ou hors role
